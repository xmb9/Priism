# Priism
Portable recovery image installer/shim manager based on [SH1MMER Legacy](https://github.com/MercuryWorkshop/Sh1mmer)

# What works? What doesn't?
Recovery ✅<br>
Sh1mmer (Debug) ✅<br>
Booting other shims ❌<br>
